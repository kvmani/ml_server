Help & Frequently Asked Questions

What is Microstructure Super-Resolution?

Microstructure Super-Resolution is a technique that enhances the quality and resolution of microstructural images using advanced image processing algorithms.

How does it work?

The system processes your input images using specialized algorithms to improve clarity, detail, and resolution without the need for deep learning.

What is EBSD Clean-Up?

EBSD Clean-Up is a tool designed to improve the quality of Electron Backscatter Diffraction (EBSD) data by removing noise and artifacts.

What file formats are supported?

The system supports common EBSD data formats including .ang, .ctf, and other standard formats.

How do I get started?

Simply select one of the available tools from the home page and follow the instructions for uploading and processing your data.

What are the system requirements?

Our tools are web-based and work with any modern web browser. No special software installation is required.

Is my data secure?

Yes, we take data security seriously. All uploaded files are processed securely and are not stored permanently on our servers.
